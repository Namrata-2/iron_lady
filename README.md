## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional but recommended)

### Installation
1. Clone or download the project files
2. Navigate to the project directory
3. Open `index.html` in your browser for the main website
4. Open `admin.html` in your browser for the admin portal

### Local Development
For optimal development experience, use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Using PHP
php -S localhost:8000
```
